LWR Project Creator
=====

This simple tool generates a controller for the Kuka LWR4+ at ISIR.

#### GUI
```bash
lwr_create_pkg
```
#### Console version
```bash
lwr_create_pkg --help
```
Example :
```bash
lwr_create_pkg rtt_lwr_sin_controller
# Use -c or --class_name SinController to override the class name.
```
